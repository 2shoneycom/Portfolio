# 1. React를 사용하는 이유

---

## React JS는 UI를 Interactive하게 만들어준다!

1. React는 간단한 방법으로 웹사이트에 상호작용을 만들어준다. 
2. 그냥 쌩 Javascript로 구현하면 React로 구현했을때보다 훨씬 불편하고 비효율적이다.

## 리액트를 선택하면 좋은 이유

1. 대부분의 웹사이트가 React로 만들어지고 있음
(에어비엔비, 페이스북, 넷플릭스 등의 회사가 사용 중, 한번 채택된 프론트엔드 기술은 웬만하면 바뀌지 않음)
2. Facebook의 지원을 받고 있음
3. 생태계가 엄청 거대함
(React는 결국 Javascript에서 왔기 때문에 JS의 생태계를 그대로 가져오게 됨)
